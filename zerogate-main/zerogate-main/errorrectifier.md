## 1. Environment & Infrastructure Errors (BLOCKING)

### ❌ ERR-001: Python Version Mismatch
- **File:** `codebase_rag/tests/conftest.py:10`
- **Error:** `ImportError: cannot import name 'Self' from 'typing'`
- **Root Cause:** System Python is 3.10 but project requires 3.12+ (`typing.Self` was added in 3.11). The `.venv` uses Python 3.12.13 via `uv`, but raw `pytest`/`python` calls bypass it.
- **Fix:** Use `uv run` for all python commands. The `Makefile` is already configured to use `uv run`. Dependencies are being installed via `uv sync --all-extras`.
- **Status:** ✅ Fixed (pending test execution)

### ❌ ERR-002: Ollama Service Offline
- **Config:** `.env` lines 4-10 (`ORCHESTRATOR_PROVIDER=ollama`, `CYPHER_PROVIDER=ollama`)
- **Error:** `curl http://localhost:11434/api/tags` → `Unable to connect to the remote server`
- **Root Cause:** Ollama is not running. The Cypher generator and RAG orchestrator cannot initialize.
- **Fix:** Run `ollama serve` in a separate terminal. Models `llama3.2` and `codellama` are currently being pulled.
- **Status:** ⏳ In Progress

### ❌ ERR-003: Memgraph Database Offline
- **Config:** `.env` lines 68-78 (`MEMGRAPH_HOST=localhost`, `MEMGRAPH_PORT=7687`)
- **Error:** `TCP connect to (127.0.0.1 : 7687) failed`
- **Root Cause:** Docker container for Memgraph is not running, and Docker Desktop is not open.
- **Fix:** Docker Desktop was started and containers `memgraph` and `lab` were spun up via `docker compose up -d memgraph lab`.
- **Status:** ✅ Fixed

## 2. Architectural Gaps (Blueprint vs. Reality)

### ❌ GAP-001: No GitHub App Integration
- **Blueprint Requirement:** Native GitHub App with webhook interception, repo cloning, delta processing
- **Current State:** Only supports manual ZIP upload via `POST /api/projects/upload`
- **Affected Files:** `codebase_rag/api/routes/projects.py`
- **Impact:** Cannot trigger scans on `git push`, no automated PR generation

### ❌ GAP-002: No Semgrep Integration (Scanner Agent)
- **Blueprint Requirement:** Semgrep as the primary vulnerability seed detector
- **Current State:** `security_queries.py` uses 7 hardcoded regex-based Cypher queries
- **Affected Files:** `codebase_rag/security_queries.py`, `codebase_rag/auto_scanner.py`
- **Impact:** Limited vulnerability detection accuracy, no custom ruleset support

### ❌ GAP-003: No LangGraph Multi-Agent Orchestration
- **Blueprint Requirement:** Scanner → Retriever → Analyst → Patcher → Verifier pipeline via LangGraph
- **Current State:** Monolithic `AutoScanner.run_full_scan()` does everything in one class
- **Affected Files:** `codebase_rag/auto_scanner.py`
- **Impact:** No composable agent workflow, no state graph, no retry loops

### ❌ GAP-004: No AST-Surgical Patching
- **Blueprint Requirement:** "Surgical AST Node Replacement — swapping out the broken logic block in the AST"
- **Current State:** `file_editor.py` uses `diff_match_patch` for string-based replacement
- **Affected Files:** `codebase_rag/tools/file_editor.py`
- **Impact:** Patches may break syntax. No structural integrity validation after patching.

### ❌ GAP-005: No Blast Radius Graph Traversal
- **Blueprint Requirement:** "The graph instantly maps the blast radius to every downstream microservice"
- **Current State:** `auto_scanner.py` only returns the directly-affected function, not the call chain
- **Affected Files:** `codebase_rag/auto_scanner.py`, `codebase_rag/security_queries.py`
- **Impact:** No multi-hop dependency chain for vulnerability impact analysis

### ❌ GAP-006: No Sandbox Verifier
- **Blueprint Requirement:** "Drops the patched AST into an OpenSandbox, compiles it, and runs unit tests"
- **Current State:** Fixes are generated by LLM but never validated before presenting to the user
- **Affected Files:** None (entirely new capability)
- **Impact:** Zero trust in AI-generated patches

### ⚠️ GAP-007: Skeletal Frontend UI
- **Blueprint Requirement:** "Luxury Minimalist" — deep dark mode, JetBrains Mono, neon accents, interactive graph
- **Current State:** Basic React+Vite with functional but unstyled pages
- **Affected Files:** `frontend/src/index.css`, `frontend/src/pages/*.jsx`
- **Impact:** Does not match the premium UX vision
